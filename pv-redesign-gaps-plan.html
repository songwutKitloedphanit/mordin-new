GAP ANALYSIS & ACTION PLAN

แผนเปรียบเทียบและปรับปรุง UX/UI ส่วน Mordin Private

เปรียบเทียบสถานะหน้าจอของแอปพลิเคชันระบบจัดการภายใน (React App) กับสติกเกอร์ต้นแบบดีไซน์ในไฟล์ ui-redesign-mockup.html พร้อมรายละเอียดส่วนต่างที่ต้องปรับแต่งเพิ่มเติมและขั้นตอนเชิงลึกในการพัฒนาต่อยอด

สรุปสถานะการดำเนินงานปัจจุบัน (Rounds 1 - 6)

ระบบได้รับการปรับโครงสร้างสไตล์ชีตและการแสดงผลกลางผ่านไฟล์ index.css รวมถึงโครงหลักของแอปพลิเคชัน (Sidebar, Header, Layout) ไปแล้วกว่า 90% ตามเป้าหมายของ UI_REDESIGN_PLAN.md ซึ่งช่วยให้ฟอร์ม ตาราง ปุ่ม ลิงก์ และสเต็ปการทำรายการแสดงผลสอดคล้องกันทุกหน้าจอโดยไม่ส่งผลกระทบต่อ logic การประมวลผลหรือโครงสร้างฐานข้อมูลเดิม

ธีมสีน้ำเงินมิตรผล (#005092)
ปรับใช้เป็นชุดสีหลักทั่วทั้งแอปพลิเคชันรวมถึงสถานะของ Badge และปุ่มกดสำเร็จรูป
โครงสร้างเมนูแบนเรียบ (Flat Menu)
Sidebar แสดงรายการเมนูครบถ้วนทันที แบ่งกลุ่มหัวข้อการทำงานชัดเจนพร้อมโลโก้
Mobile Responsive Overrides
ตารางกรอบข้อมูลและปุ่มควบคุมขยายขนาด touch-targets และรองรับ horizontal-scroll บนมือถือ

ตารางเปรียบเทียบองค์ประกอบ (Mockup vs. Codebase)

องค์ประกอบดีไซน์ ต้นแบบดีไซน์ (ui-redesign-mockup.html) สถานะในโค้ดปัจจุบัน (React App) ประเมินผล
Sidebar Logo & Menu
หัวแถบเมนูด้านซ้ายและโลโก้หลัก
 โลโก้ MITR PHOL ในกรอบโค้งมนพื้นหลังขาว, เมนูกลุ่มแบนราบ, รายการ active มีเส้นขอบสีทองด้านซ้าย เหมือนต้นแบบ (Sidebar.tsx ได้รับการเขียนใหม่ด้วย Tailwind/JSX) เรียบร้อย
Sidebar User Footer
โปรไฟล์ผู้ใช้ด้านล่างเมนูซ้าย
 มีบล็อกโปรไฟล์ผู้ใช้ (รูปย่อ, ชื่อจริง, บทบาท, ปุ่มไอคอน Logout) ด้านล่างสุดของ Sidebar ไม่มีส่วนนี้ใน Sidebar, ข้อมูลผู้ใช้และปุ่มออกจากระบบถูกย้ายไปรวมอยู่ที่เมนู dropdown ด้านบนมุมขวา (Header.tsx) รอปรับปรุง
Topbar Search Input
กล่องค้นหาหลักบนแถบด้านบน
 มีกล่องค้นหาแบบจำลอง ⌘K อยู่ตรงกลาง Topbar สำหรับค้นหาแปลง ชาวไร่ หรือตัวอย่างแบบด่วน ไม่มีกล่องค้นหากลางบน Header, ฟังก์ชันค้นหาแยกใช้ระดับโมดูลในตารางของแต่ละหน้าโดยเฉพาะ รอปรับปรุง
Dashboard KPIs & Insights
การ์ดสถิติเด่นและคำแนะนำปุ๋ย
 KPI การ์ดแสดงผลขอบสีซ้าย, มีกล่อง Insights 3 คอลัมน์แสดงข้อมูลสรุปธาตุอาหาร/สูตรปุ๋ยเด่น การ์ดสถิติหลักปรับสไตล์และขอบสีแล้วตาม R5, ยังไม่มีแถบแสดงกล่อง Insights 3 คอลัมน์ด้านล่างการ์ดสถิติ มีบางส่วน
Soil Nutrient Analysis Tabs
กราฟสถิติแสดงผลดินรายธาตุ
 มีปุ่มสลับแท็บ (N, P, K, pH, OM) แสดงแถบระดับความอุดมสมบูรณ์เป็นเปอร์เซ็นต์สะสมแบบนอน ใช้กราฟวงกลม/กราฟแท่งแบบไดนามิกเต็มรูปแบบด้วย ECharts/Chart.js ซึ่งยืดหยุ่นกว่าข้อมูลสถิติจำลอง ใช้ระบบจริง
Sample Status Stepper Timeline
ขั้นตอนสเต็ปบนหน้ารายละเอียดรับดิน
 แสดงจุดสเต็ป 5 ขั้นตอน (เก็บตัวอย่าง > รับเข้า > แล็บ > ประมวลผล > ออกรายงาน) บ่งชี้สถานะตัวอย่างดินที่สแกนล่าสุด แสดงผลข้อมูลในรูปแบบตารางรายละเอียด (InfoTable) และ Badge กำกับสถานะ ยังไม่มีสเต็ปไทม์ไลน์ 5 ขั้นตอนที่วาดด้วย CSS รอปรับปรุง
Lab Result Batch Sheet
ตารางป้อนข้อมูลผลแล็บ
 ตารางป้อนค่าแล็บแบบทีละตัวอย่าง ช่องป้อนคอลัมน์ pH เกินขอบเขตมีกรอบแดง และมีป้ายกำกับสถานะการตรวจสอบซ้ำ ใช้ spreadsheet ของ Handsontable บน Desktop มีการลงสี highlight ข้อมูลผิดปกติ และเพิ่มป้ายแจ้งเตือนผู้ใช้มือถือให้ไปกรอกแบบเดี่ยว เรียบร้อย

ขั้นตอนการดำเนินการเพิ่มเติม (Gaps Action Plan)

เพื่อให้ UI ของแอปพลิเคชันจริงตรงตามต้นแบบจำลอง 100% มีงานปรับแต่งโครงสร้าง React และ JSX 4 รายการที่สามารถเพิ่มเข้าไปในโค้ดปัจจุบันได้ทันที ดังนี้:

1
เพิ่มแถบข้อมูลผู้ใช้ (sb-foot) ที่ส่วนท้ายของ Sidebar.tsx

ดึงข้อมูลผู้ใช้ปัจจุบันจาก useAuth() เข้าไปแสดงที่ด้านล่างสุดของ Sidebar พร้อมปุ่มเรียกใช้งาน logout() โดยตรง เช่นเดียวกับ mockup

ไฟล์ที่ต้องแก้ไข: src/components/layout/admin/Sidebar.tsx 
// เพิ่มโค้ด HTML/JSX ด้านล่างสุด ก่อนปิดแท็ก </aside>
const { user, logout } = useAuth();
const displayName = [user?.firstName, user?.lastName].filter(Boolean).join(' ') || user?.username || 'ผู้ใช้งาน';
const avatarInitial = displayName.charAt(0).toUpperCase();

return (
  <aside className=...>
    {/* เนื้อหา Sidebar Nav เดิม */}
    
    {/* Sidebar Footer (sb-foot) */}
    <div className="sb-foot d-flex align-items-center gap-2 px-3 py-3 border-top border-white/10" style={{ background: 'rgba(0, 0, 0, 0.15)' }}>
      <div className="sb-avatar rounded-circle bg-white text-primary d-flex align-items-center justify-content-center fw-bold" style={{ width: 34, height: 34, color: '#005092' }}>
        {avatarInitial}
      </div>
      <div className="flex-grow-1 min-w-0">
        <div className="text-white fw-bold text-truncate" style={{ fontSize: 13 }}>{displayName}</div>
        <small className="text-white-50" style={{ fontSize: 11 }}>{user?.role}</small>
      </div>
      <button 
        onClick={() => logout()} 
        className="btn btn-link p-0 text-white-50 hover-text-danger border-0"
        title="ออกจากระบบ"
      >
        <i className="fas fa-right-from-bracket" />
      </button>
    </div>
  </aside>
);
